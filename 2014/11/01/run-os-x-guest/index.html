在OS X上运行OS X

Nov 01, 2014

有没有想过在Mac OS X操作系统中再运行一个OS X系统？

例如，在最新的OS X 10.10 Yosemite的虚拟机中再运行一个Yosemite，甚至运行各种OS X版本。也许你觉得没有必要，但是，对于开发和测试来说，需要在不同的操作系统环境中验证程序运行。对于我个人来说，我也希望能够在一个纯粹的环境中运行X Server，来体验一些企业级的管理工具。

虚拟机软件

我购买的Parallels Desktop 10最新版本终于支持在Yosemite操作系统中运行Yosemite虚拟机（没有升级前虚拟机操作系统会反复进入安装模式无法正常启动）

安装主要有以下几种方式（根据回忆记录，可能有差异）：

  • 从物理服务器recovery区域启动，然后直接联网下载安装镜像进行安装
  • 从物理服务器recovery区域启动，然后使用物理主机的TimeMachine备份进行recovery（可以恢复成和物理主机一模一样的虚拟机）
  • 使用OS X的USB安装盘（我是从下载的Yosemite安装包转制出USB安装盘）安装
  • 使用Parallels工具，将下载的Yosemite安装包转制成安装iso镜像进行安装

整个过程和物理主机安装（或者TimeMachine恢复）没有什么区别，我除了遇到因为Parallels早期版本10不能很好支持Yosemite虚拟机的问题（等待了几天就发布了一个修复版本），其他都比较顺利。

在Parallels虚拟机中运行OS X也支持TimeMachine备份，也就是说可以很方便地制作各个阶段虚拟机的在线备份，方便做测试时弄乱环境后及时恢复。
